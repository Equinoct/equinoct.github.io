Tidal Calender

The Tidal Calender application allows residents of coastal regions affected by tidal flooding to upload relevant data and media. This data will be used for research and co-creating community solutions for the worsening tidal situation we are faced with.

Currently, only the android version of the application is in existence. The web and iOS versions are under development, and will be available at this website soon. You will be required to use an email address to register your account on the application.

Instructions

Kindly read instructions before you proceed. The app is not available on Google Play Store yet, so you need to follow certain steps to 'Allow App Installation fron Unknown Sources' to install the app. As soon as it is available, we shall update the page with the Play Store link to the application.

  1. Click on the 'Download Tidal Calender APK' button below. The .apk file will get downloaded to your device.
  2. Now, you can install the application either through the downloads within your browser, for ex. Chrome, or through your file manager application.
  3. You will need to allow "Install unknown apps" for that application. For this, Open Settings > Apps > Special App Access. 'Special App Access' may be within 'Advanced' under Apps.
  4. Tap on 'Install Unknown Apps' and then on the app that you are going to use to install the Tidal Calender application. Enable the toggle next to "Allow from this source".
  5. Go to the relevant application, i.e., Chrome / File Manager and click on the tidal_calender.apk file. The file will appear under 'downloads' in Chrome / File Manager. Follow the onscreen prompts to install the Tidal Calender application.
  6. In case you have a device with different settings / android version, kindly go to this link where you will be able to find detailed instructions on allowing install from unknown sources.

The data collected through this program will be used only for research and co-creating community solutions.
