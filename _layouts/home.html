Tidal Calender

Only the android version of the application is in existence. The web and iOS versions are under development, and will be available at this website soon.

Instructions

The app is not available on Play Store yet, so you need to follow certain steps to 'Allow App Installation fron Unknown Sources' to install the app. We shall update the page with the Play Store link to the application soon.

  1. You can install the application either through the downloads within your browser, for ex. Chrome, or through your file manager application.
  2. You will need to allow "Install unknown apps" for that application. For this, Open Settings > Apps > Special App Access. 'Special App Access' may be within 'Advanced' under Apps.
  3. Tap on 'Install Unknown Apps' and then on the app that you are going to use to install the Tidal Calender application. Enable the toggle next to "Allow from this source".
  4. Go to the relevant application, i.e., Chrome / File Manager > downloads and click on the tidal_calender.apk file. Follow the onscreen prompts to install the Tidal Calender application.
  5. In case you are unable to find these settings, kindly go to this link where you will be able to find detailed instructions on allowing install from unknown sources.
